
# TRYONYOU–ABVETOS–ULTRA–PLUS–ULTIMATUM

Plataforma integral de moda inteligente basada en la **Patente Ultimátum**.

**Flujo patentado (resumen):**
1. **Cuestionario Tryon** → datos personales (peso, talla, ciudad), gustos (estilos, colores, gramaje).
2. **Avatar 3D + Biometría** → 4 fotos con folio A4 + mano (360º) → registro iris + voz.
3. **Tienda (PAU)** → recomendaciones y **clicks** del usuario (aquí se trackea).
4. **FGT Top-20 Google** → tendencias globales mezcladas con gustos/clicks.
5. **Mockups** → 20/ref (3 por pantalla, scroll/paginación).
6. **Checkout** → compra **automática de prenda blanca** + **impresión mockup** (CAP+LiveIt JIT, trazabilidad).
7. **Armarios** → Inteligente (registro) y Solidario (donación/2ª mano) + AutoDonate.
8. **Pago AVBET** → validación dual **iris + voz**.
9. **EPCT** → **watcher** + **solicitud de pruebas gratuitas** para nuevas invenciones.
10. **Bot ABVET** → orquestador (avisos, despliegues, recordatorios).

**Dominios**: tryonme.com · tryonyou.com · abvetos.com/abvet.com · liveitfashion.com · vvlart.com
